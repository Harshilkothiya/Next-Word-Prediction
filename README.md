# 📝 Next Word Prediction

> *Teaching machines to understand context—one word at a time.*

This project implements a **Next Word Prediction** model capable of predicting the most probable next word based on a given text sequence. By learning language patterns from training data, the model captures contextual relationships between words to generate intelligent predictions.

---

## 💡 Why Next Word Prediction?

Whenever you type on your smartphone or use AI writing assistants like **ChatGPT**, **Gemini**, or **Google Keyboard**, a language model predicts what you're likely to type next.

This project demonstrates the fundamental idea behind those systems using Deep Learning.
